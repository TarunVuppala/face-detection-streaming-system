# Real-Time Face Detection Video Streaming System

Containerized FastAPI and React application for receiving a live browser video feed, detecting a face, drawing a ROI rectangle without OpenCV, storing ROI metadata, and displaying the annotated stream.

## Development Status

This project is being built incrementally. The intended stack is:

- FastAPI
- PyAV
- MediaPipe
- Pillow
- PostgreSQL
- SQLAlchemy
- React
- Docker Compose
